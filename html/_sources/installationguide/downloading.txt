=========================
Downloading Humdrum
=========================


The Github Repos
---------------------

Installing Humdrum
~~~~~~~~~~~~~~~~~~~~~~
At the moment, the best way to install both the Humdrum 
toolkit and the Humdrum extras is to use **git**. 

Pedro Kroger's github repository works best for the standard
Humdrum toolkit:

``git clone https://github.com/kroger/humdrum.git``


Installing Humdrum Extras
~~~~~~~~~~~~~~~~~~~~~~~~~~~

Craig Sapp's repo of his Humdrum Extras can be found here:

``git clone https://github.com/craigsapp/humextra.git``
