To compile the source into html pages, you need to install sphinx.  
Try these installation steps to install it:


* Install sphinx with pip:
	sudo pip install sphinx

* If you have problems, try installing pip:
	sudo easy_install pip

* Check to see if installed:
	which sphinx-build
  which should reply something like this:
	/usr/local/bin/sphinx-build

=================================================

Installing theme (https://github.com/snide/sphinx_rtd_theme)


* Download the RtD theme for Sphinx:
	sudo pip install sphinx_rtd_theme
