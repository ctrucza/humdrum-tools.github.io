Humdrum Exercise #8 - Answers

All of the following commands assume that you are located in the appropriate score directory.

6.

In the Dutch folksong "Welekomen Keremisvolk" (neder012.krn) which phrase contains the most notes?
COMMAND:     yank -o { -e } -r 1 neder012.krn | census -k
                            yank -o { -e } -r 2 neder012.krn | census -k
                            yank -o { -e } -r 3 neder012.krn | census -k
                            yank -o { -e } -r 4 neder012.krn | census -k
ANSWER:         3rd phrase (9 notes)
