COMMAND

 midreset -- reset MIDI controller card

SYNOPSIS

 midreset

DESCRIPTION

 The midreset command resets the MIDI controller card. The midreset command is useful if a MIDI application has terminated abnormally -- leaving the MIDI card unaccessable to further use.

OPTIONS

 The midreset command provides only a help option:
-hdisplays a help screen summarizing the command syntax
Options are specified in the command line.

PORTABILITY

 DOS 2.0 and up.

SEE ALSO

 perform (4), smf (4), tacet (4)
