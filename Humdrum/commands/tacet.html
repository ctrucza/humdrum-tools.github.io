COMMAND

 tacet -- reset MIDI channels to ensure silence

SYNOPSIS

 tacet [-i hex]

DESCRIPTION

 The tacet command resets the MIDI output. It sends "all-notes-off" commands on MIDI channels 1-16. The tacet command is useful if a MIDI application has terminated abnormally -- leaving one or more sounding notes ("ciphers") on a MIDI device.

OPTIONS

 The tacet command provides the following options:
-hdisplays a help screen summarizing the command syntax
-i hexassign MIDI input/output address to hex
Options are specified in the command line.

The -i option is used to specify the input/output address of the MIDI card. The default address is `330.' The address is specified as a hexadecimal number.

PORTABILITY

 DOS 2.0 and up.

SEE ALSO

 midreset (4), perform (4), smf (4)
